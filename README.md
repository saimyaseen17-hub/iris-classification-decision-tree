🌸 Iris Dataset - Decision Tree Classification

This project demonstrates a simple Machine Learning model using a Decision Tree Classifier on the Iris dataset.

📊 Dataset
The dataset contains measurements of iris flowers:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Target)

🎯 Objective
To classify iris flower species:
- Setosa
- Versicolor
- Virginica

🧠 Algorithm Used
- Decision Tree Classifier (Scikit-learn)

⚙️ Steps Followed
1. Data loading using pandas
2. Feature selection (X) and target selection (y)
3. Train-test split
4. Model training
5. Prediction
6. Accuracy evaluation

📈 Result
Model achieved good accuracy on test data.
